Just click and wait to try the service

The website owner can offer a trial to visitors after verifying the user.

Very Easy Sign-up

Your clients can try out your products without having to bother with the Sign-up process. Once they Sign-up, they can try your products on other sites with one click Sign-in. Clients need to create a solana address in advance starting with "VV..."(two V charactor). It takes several minutes to create an address starting with "VV...", and the client must pay the CPU cost. This is the same as mining Bitcoin. This prevents attackers from creating a large number of accounts and trying your prototype over and over again. Many current sign-up systems use CAPTCHAs or phone numbers to prevent from attackers. However, CAPTCHAs are cumbersome and submitting a phone number risks exposing personal information.

Server-side implementation is also easy

The code on the left is the entire sample code using Node, with the web3verifier embedded in test.html, and authentication from the client-client is done by the verify function. If the authentication fails, ok.html is displayed; if not, ng.html is displayed.

import * as https from 'https'
import * as fs from 'fs'
import { Request, Verifier, Verify_Request_Param } from './verifier';
import assert from 'assert';

let key  = fs.readFileSync('ssl/key.pem')
let cert = fs.readFileSync('ssl/cert.pem')
const options = { key, cert }

let verifier = new Verifier( [] )

let server = https.createServer(options, async function(request, response) {

    console.log( request.url )

    let rtnHtml = "not_found"
    let staticHtml   = fs.readFileSync('test.html').toString()
    let verifyOkHtml = fs.readFileSync('ok.html').toString()
    let verifyNgHtml = fs.readFileSync('ng.html').toString()


    if ( request.url === '/test.html' ){
        rtnHtml = staticHtml
    } else {
        assert( request.url !== undefined )

        if ( verifier.isRequestForPrepare(request.url) ){
            rtnHtml = verifier.response( request.url )
        } else {
            if ( verifier.isRequestVerify(request.url) ){
                if ( await verifier.verify( request.url ) == true ){
                    console.log("  verify() === true")
                    rtnHtml = verifyOkHtml
                } else {
                    console.log("  verify() === false")
                    rtnHtml = verifyNgHtml
                }
            }
        }

    }

    response.writeHead(200)
    response.end(rtnHtml)
})

server.listen(443, () =>{
    console.log("server start")
})

Payment can be made using Web3Verifier.

Web3Verifier also functions as a client's wallet because web3verifier keep client secretkey. After the client pays the specified amount of money, seller can verifies the payment and provides the service.
